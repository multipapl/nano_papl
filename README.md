# Nano Papl | AI Archviz Automation

Nano Papl is a specialized AI-powered tool designed to streamline Architectural Visualization workflows. It leverages the **Google Gemini 3 Pro** model to generate high-quality prompts and process image-to-image transformations.

## Key Features

- **Prompt Constructor**: Generate complex prompt matrices for architectural scenes (seasons, lighting, context).
- **Batch Studio**: Automate image-to-image processing for multiple folders and projects.
- **Ad-hoc Chat**: Interactive AI assistant with vision support and smart history management.
- **Standalone EXE**: Fully portable desktop application built with PySide6.

## Requirements

- Google Cloud API Key (Gemini API)
- Windows 10/11

## How to use

1. Run `Nano Papl.exe` from the `dist` folder.
2. Go to **Settings** and enter your Google API Key.
3. Use the **Constructor** to create `prompts.md` for your projects.
4. Use **Batch Studio** to start the automated rendering process.
